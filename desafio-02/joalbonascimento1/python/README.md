Função numeroPrimo() recebe um número e verifica a quantidade de divisões possíveis para aquele número, caso sejam.
Comecei a estudar como estruturas de dados podem melhorar o desempenho de aplicações e lembrei que essa função "crua" calcularia literalmente de 1 a 10000. Para melhorar o desempenho eu coloquei uma condicional que quando ultrapassasse o número em estudo, ele parasse o loop para economizar recursos.

Para executar a função digite no terminal e dentro da pasta "joalbonascimento/python": python desafio02.py
