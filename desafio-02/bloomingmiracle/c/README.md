# Desafio 02 - Números Primos

Programa em C que lista números primos entre 1 e 10000 utilizando o Crivo de Eratóstenes.
## Compilação

### Linux

```bash
gcc primos.c -o primos
```

### Windows (MinGW)

```bash
gcc primos.c -o primos.exe
```

## Execução

### Linux

```bash
./primos
```

### Windows

```bash
primos.exe
```
