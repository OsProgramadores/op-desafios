## Desafio 2

### A classe MeuPrograma funciona da seguinte forma:

1: O laço começa em 0 e termina em 10000

2: Um contador é inicializado com 0

3: Outro laço é percorrido verificando a quantidade de divisores do número atual

4: Se a quantidade de divisores for 2, ele é primo, então é printado

5: Se não, não é primo, logo não é printado
