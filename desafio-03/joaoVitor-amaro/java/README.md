# Desafio 3 - Números Palidrômicos

## Solução

Foi criado uma função que converte os numeros em string
para fazer a comparação entre o número original com o invertido 
para verificar se é palidrômico e será imprimido
esse número palidrômico.

## executar

```bash
javac java.Palindromo.java
java java.Palindromo
```

Foi utilizado o Java 21.
