# Desafio 03 - Números Palindrômicos

## Descrição
Um programa que imprime todos os números palindrômicos entre dois outros números.

## Solução
O programa recebe os dois números do intervalo como argumento e valida as entradas. Em seguida, itera sobre o intervalo e inverte cada número usando operações matemáticas. Se o número invertido for igual ao original, ele é impresso.

## Como executar
```bash
javac ListaPalindromos.java
java ListaPalindromos <numeroInicial> <numeroFinal>
```
Testado com Java 21.
