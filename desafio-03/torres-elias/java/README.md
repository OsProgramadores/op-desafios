# Desafio 03 - Números Palindrômicos

## Descrição
Um programa que imprime todos os números palindrômicos entre dois outros números.

## Solução
A abordagem solicita os dois valores do intervalo ao usuário, verifica se eles são válidos e itera sobre eles, invertendo cada candidato usando operações matemáticas.

## Como executar
```bash
javac ListaPalindromos.java
java ListaPalindromos
```
Testado com Java 21.
