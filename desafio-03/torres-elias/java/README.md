# Desafio 03 - Números Palindrômicos

## Descrição
Um programa que imprime todos os números palindrômicos entre dois outros números.

## Solução
A abordagem solicita os dois valores do intervalo ao usuário e itera sobre eles, convertendo cada candidato para String e comparando com seu inverso gerado via `StringBuilder.reverse()`.

## Como executar
```bash
javac ListaPalindromos.java
java ListaPalindromos
```
