Números Palíndromos

## Descrição
Programa que encontra números palíndromos entre dois valores informados pelo usuário.

Como o código funciona ?
---------------------------------------------------------------------
Passo 1: Receber os números
O programa pede dois números: o inicial e o final.

Passo 2: Verificar cada número
Para cada número entre o inicial e o final, o programa:
- Converte o número em texto
- Inverte o texto 
- Compara o texto original com o invertido pra ver se é palindromo mesmo

Passo 3: Guardar o resultado
Se forem iguais, o número é palíndromo e é adicionado a lista.

Passo 4: Mostrar o resultado
O programa exibe todos palíndromos encontrados.
---------------------------------------------------------------------

Como executar:
```bash
python palindromos.py
