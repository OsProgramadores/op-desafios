# Desafio 08

  ## Informações sobre a implementação

  Esta solução lê frações pela entrada padrão ou por arquivo.

  O programa valida o formato da entrada, calcula o MDC entre
  numerador e denominador e simplifica a fração.

  Quando possível, a saída é mostrada como número inteiro ou número
  misto.

  ## Como rodar o exercício

  Entre na pasta da solução:

  ```bash
  cd desafio-08/ju-caju/python

  Rodando pela entrada padrão:

  python3 resposta.py

  Rodando com arquivo:

  python3 resposta.py arquivo.txt

  ## Como instalar o Python

  Verifique se o Python 3 já está instalado:

  python3 --version

  Se não estiver instalado, no Ubuntu ou Debian rode:

  sudo apt update
  sudo apt install python3
